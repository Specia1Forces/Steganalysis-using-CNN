Тему выпускной квалификационной работы в бакалавриате посвятил обнаружению стеганографических вставок в цифровых изображениях. В рамках работы разработал гибридную систему анализа изображений для выявления скрытых данных с использованием как классических статистических методов, так и нейронных сетей.

— применил RS-метод для обнаружения LSB-вставок;
— использовал CNN на базе ResNet50 для выявления более сложных вставок семейства Koch;
— самостоятельно сформировал и подготовил датасет изображений;
— дообучил ResNet50 под задачу бинарной классификации;
— достиг accuracy 0.995 и F1-score 0.996 на валидационном наборе данных.

Основная идея системы заключалась в комбинировании статистических методов для простых случаев и CNN-подходов для более сложных типов стеганографических вставок.
